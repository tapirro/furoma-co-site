# Furoma

> Suplemento para el control de la presión arterial con espino blanco, alcachofa, jengibre y magnesio.

**[Visitar furoma.co](https://furoma.co)**

## Sobre este proyecto

Este repositorio contiene el sitio web de [Furoma](https://furoma.co), un recurso de salud basado en evidencia científica enfocado en **Presión Arterial**.

### Ingredientes clave

Espino blanco (*Crataegus monogyna*), extracto de alcachofa, jengibre, magnesio bisglicinato

## Tecnología

- HTML5 + CSS3 (sitio estático)
- Hospedado en GitHub Pages
- Sin dependencias de JavaScript frameworks
- Optimizado para Core Web Vitals

## Evidencia científica

Todo el contenido está respaldado por:
- Estudios clínicos publicados en PubMed
- Monografías de la EMA (European Medicines Agency)
- Guías de la OMS sobre plantas medicinales
- Base de datos TRAMIL de plantas del Caribe y Latinoamérica
